|---|---------|-------------|

## рҹҺҜ How to Use

Once launched, you'll see the **Library Dashboard** menu:

```
===== Library Dashboard =====
1. Load Data
2. Calculate Statistics
3. Filter Transactions
4. Generate Report
5. Visualize
6. Exit
Enter choice: _
```

**Step-by-step walkthrough:**

1. **Start with `1` вҶ’ Load Data** вҖ” Point the system to your transaction CSV or dataset.
2. **Use `2` вҶ’ Calculate Statistics** вҖ” Instantly get summaries: total books, active members, overdue items.
3. **Try `3` вҶ’ Filter Transactions** вҖ” Apply custom filters to isolate specific records.
4. **Run `4` вҶ’ Generate Report** вҖ” Produce a formatted text or CSV report for records.
5. **Explore `5` вҶ’ Visualize** вҖ” See your library data come alive in charts.
6. **Press `6` вҶ’ Exit** вҖ” Safely quit the program.
